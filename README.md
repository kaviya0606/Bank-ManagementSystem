# 🏦 Bank Management System

## 📌 Project Overview

The Bank Management System is a console-based application developed using C++. This project simulates basic banking operations and demonstrates the use of Object-Oriented Programming (OOP) concepts and File Handling for persistent data storage.

The system allows users to create bank accounts, perform deposits and withdrawals, check account balances, and save customer records to a file.

---

## 🎯 Objectives

* To implement core banking operations using C++.
* To apply Object-Oriented Programming concepts.
* To store and manage customer records using File Handling.
* To provide a simple and secure banking management solution.

---

## ✨ Features

* Create New Account
* Display Account Details
* Deposit Money
* Withdraw Money
* Balance Inquiry
* Save Customer Records to File
* Input Validation for Transactions
